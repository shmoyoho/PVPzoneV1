# PvpZone

Плагин для Minecraft (Spigot/Paper 1.21) — PvP зоны с рейтингом, дуэлями и магазином бонусов.

**Версия:** 1.1.1  
**Автор:** Galyakyxnya  
**API:** 1.21

---

## Возможности

- **PvP зоны** — задаются двумя точками (`/pvpz1`, `/pvpz2`). В зоне действует отдельный набор (кит), инвентарь снаружи сохраняется.
- **Рейтинг и очки** — за убийства в зоне начисляются рейтинг и очки для магазина; киллстрики дают бонус.
- **Дуэли** — вызов игрока на дуэль (`/pvp <игрок>`), приём/отклонение (`/pvpaccept`, `/pvpdeny`).
- **Магазин** — покупка бонусов за очки: здоровье, скорость, прыжок, урон (`/pvpshop`).
- **Наборы (киты)** — создание наборов из инвентаря (`/pvpkit`), привязка к зоне (`/pvpsetkit`).
- **Защита от дюпа** — при смерти в PvP зоне предметы не выпадают (инвентарь сохраняется).
- **Топ игроков** — таблица лидеров по рейтингу (`/pvptop`).

## Установка

1. Скачайте `PvpZone-1.1.1.jar` (или соберите из исходников: `mvn clean package`).
2. Поместите JAR в папку `plugins` сервера.
3. Перезапустите сервер. В `plugins/PvpZone/` появятся `config.yml` и база SQLite.

---

## Сборка из исходников

```bash
git clone https://github.com/shmoyoho/PVPzoneV1.git
cd PVPzoneV1
mvn clean package
```

Готовый JAR: `target/PvpZone-1.1.1.jar`.

**Требования:** Java 17+, Maven 3.6+

## Команды

| Команда | Описание | Права |
|--------|----------|--------|
| `/pvpz1` | Установить первую точку PvP зоны | `pvpzone.admin` |
| `/pvpz2` | Установить вторую точку PvP зоны | `pvpzone.admin` |
| `/pvpkit <название>` | Создать набор из текущего инвентаря | `pvpzone.admin` |
| `/pvpkit list` | Список наборов | `pvpzone.admin` |
| `/pvpsetkit <зона> <набор>` | Привязать набор к зоне | `pvpzone.admin` |
| `/pvpzone` | Управление зонами (list, create, delete, info, reload и др.) | `pvpzone.admin` |
| `/pvptop` | Топ игроков по рейтингу | все |
| `/pvpshop` | Магазин PvP бонусов | `pvpzone.shop` |
| `/pvp <игрок>` | Вызвать на дуэль | все |
| `/pvpaccept` | Принять дуэль | все |
| `/pvpdeny` | Отклонить дуэль | все |
| `/pvpfix` | Исправить PvP состояние | все |

Алиасы: `pvpz` → `pvpzone`, `pvprating` / `pvpleaderboard` → `pvptop`, `pvpstore` → `pvpshop`.

## Права (Permissions)

- `pvpzone.use` — использование PvP зоны (по умолчанию у всех).
- `pvpzone.admin` — админ: зоны, киты, управление плагином.
- `pvpzone.shop` — доступ к магазину.
- `pvpzone.*` — все права (по умолчанию для OP).

---

## Конфигурация

Файл `plugins/PvpZone/config.yml`:

- **zone** — интервал проверки, keep-inventory, защита блоков и т.д.
- **rating** — очки за убийство/смерть, киллстрики.
- **leader-effects** — эффекты для лидеров (следы, аура).
- **bonuses** — параметры бонусов в магазине (здоровье, скорость, прыжок, урон).
- **messages** — префикс и тексты сообщений.

После изменений: `/pvpzone reload` или перезагрузка плагина.
