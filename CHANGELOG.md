# Changelog

## 1.1
- **Инвентарь:** плагин управляет инвентарём только при выходе/входе в PvP-зоне. В остальных случаях инвентарь ведёт сервер.
- Выход в зоне → при входе восстанавливается мирской инвентарь, затем при нахождении в зоне выдаётся PvP-набор; при выходе из зоны — возврат старых предметов.
- Мультиязычность (ru/en) в конфиге, все команды и сообщения через lang-файлы.
